# Screenshots

Add preview assets here for the root README.

## Suggested captures

| File | What to capture |
|------|-----------------|
| `dashboard.png` | Dashboard with stat cards, spending charts, account balances, and goals |
| `transactions.gif` | Short screen recording: filter transactions, open create/edit modal, or row actions menu |

## Tips

- Run the full stack (`tilt up` or API + `pnpm dev`) with sample data loaded
- Use a consistent viewport width (~1280px) for static screenshots
- Keep GIFs under ~10 seconds and ~2MB for GitHub README performance
- Prefer light or dark mode consistently across assets
